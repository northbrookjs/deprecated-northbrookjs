Northbrook

Manage your projects with all the power of NPM

Core Features

  • Manage project with ease
  • Useful out-of-box
  • Create meaningful commits
  • Automatic Semantic Versioning
  • Automated Changelog Generation
  • Simple Automated Deployment
  • Powerful plugin system

Northbrook is designed to ease javascript fatigue, reduce the time needed for new project setup, and introduce a means of creating reproducible, shareable and versionable configurations. Promoting best-practices along the way.
