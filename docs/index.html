Northbrook

Manage your projects with all the power of NPM

Core Features

  • - Manage projects with ease  
  • - Useful out-of-box  
  • - Create meaningful commits  
  • - Semantic versioning  
  • - Changelog generation  
  • - Automated deployment  
  • - Powerful plugin system  

Northbrook is designed to ease javascript fatigue, promote best-practices, reduce the time needed for new project setup, and introduce a means of creating reproducible, shareable and versionable configurations.
